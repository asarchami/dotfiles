# hyprland-docs

Official Hyprland Wiki reference at **https://wiki.hypr.land**.

Since Hyprland 0.55, `hyprlang` is deprecated in favor of Lua. Config file: `~/.config/hypr/hyprland.lua`. For old syntax, see the 0.54 wiki pages.

## Quick Commands
- `hyprctl version` — check version
- `hyprctl reload` — reload config
- `hyprctl monitors` — list monitors
- `hyprctl clients` — list windows
- `hyprctl binds` — active keybinds
- Wiki version selector matches wiki pages to your Hyprland release.
